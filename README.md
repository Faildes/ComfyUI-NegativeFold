# Negative Fold
*Fold Negative prompt to the Prompt for usage on Turbo models*

----
## Usage
1. Put this extension in `ComfyUI/custom_nodes`

2. Under `conditioning`, you can find `Fold Negative Into Positive (Conditioning)` node

3. Connect Positive Conditioning to `positive`, Negative Conditioning `negative`, `CONDITIONING` to the Sampler's `positive`
